github从入门到放弃

[TOC]

2019年5月26日，据《南华早报》消息，微软将停止接受华为的新订单，并且，华为笔记本电脑中的Windows操作系统和其他内容相关服务都已被微软暂停。

这一消息引发了网友对微软的声讨，另一方面，Windows的退出也令另一操作系统Linux成为了众人讨论的焦点。

标题中所提到的“神”正是Linus Torvalds，他被业内称为Linux和Git之父。

他不仅写出了Linux内核，还在短短十天内开发出Git软件，至今依然无偿开放给全世界所有人，深刻影响了整个互联网。


“有些人生来就具有统率百万人的领袖风范，另一些人则是为写出颠覆世界的软件而生。唯一一个能同时做到这两者的人，就是Linus Torvalds。”

这是美国《时代》周刊对他作出的极高评价。

Linux内核不仅是GNU操作系统的基础，同时也是安卓系统的内核。

在这个内核的基础上，才得以诞生这个世界上的绝大多数电子设备（路由器，交换机，手机，服务器）。
而Git的走红，则完全是一场意外。

在Git被创造出来之后的三四年内，业内并不看好，但好在大家都在慢慢适应。

直到2007年，使用了Git为核心技术的GitHub诞生，并成为了几乎每个开发者都会使用的、赫赫有名的开源社区。

早从2000年开始，他就为保证自己写出的这两个项目永远开放不受制，拒绝了许多公司提供的高薪职位，包括来自乔布斯的亲自邀请。

Linus直接表示自己讨厌Mac OS的Mach内核，更何况乔布斯还希望他放弃Linux开发并开始做其他内容。

2012年，Linus Torvalds获得了有“技术界诺贝尔奖”之称的千禧技术奖。

2014年，Linus Torvalds获得了计算机先驱奖，成为对世界影响最大的十大程序员之一。

比起乔布斯和比尔盖茨，他更像一位真正的自由主义者——在采访中，Linus表示只想做技术，不想处理技术以外的事情，只要赚的钱能够保证养活家庭就行了。

至于Linus为什么被程序员称为大神？

不仅因为他创造了优秀的开源软件，还因为整个开源软件的发展都是在他的倡导下兴起的。

没有Linux和Git，RMS（自由软件之父）在自由软件方面的计划只能是个空想。

所以一定程度上，整个开源软件社区之所以存在，都与Linus Torvalds分不开。
